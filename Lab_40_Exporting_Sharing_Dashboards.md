# Lab 40 — Exporting & Sharing Dashboards

## Objectives
- Understand how to export dashboards in different formats such as PDF or CSV.
- Learn to share dashboards by generating shareable links.
- Set appropriate permissions for dashboard access.
- Confirm that sharing settings allow for read-only viewing.

## Prerequisites
- Basic understanding of operating system navigation (Windows/Linux/MacOS).
- Familiarity with using open-source dashboarding tools.
- Pre-installed open-source dashboard tool (e.g., Grafana, Metabase).

---

## Task 1: Export a Dashboard

### Step 1.1: Open Your Dashboard
- Open your preferred dashboard tool (Grafana, Metabase, etc.).
- Navigate to the dashboard you wish to export.

### Step 1.2: Export as PDF
- Locate the export option in the dashboard menu (often a 'three-dot' menu).
- Select **Export** → **PDF**.
- Specify settings (page size, orientation).
- Click **Export** and save the PDF to your desired location.

### Step 1.3: Export as CSV
- If supported, select the data table to export.
- Click **Download/Export**.
- Choose CSV as the file format.
- Save the CSV to your desired location.
> **Note:** Ensure your tool’s license allows CSV export.

---

## Task 2: Share a Dashboard

### Step 2.1: Generate a Shareable Link
- Access sharing options in the dashboard.
- Select **Get Link** or **Share Dashboard**.
- Customize settings like expiry or access restrictions.
- Copy the generated link for sharing.

### Step 2.2: Set Permissions for Read-Only Access
- Navigate to **Permissions** or **Access Control** settings.
- Add a user or group (by email or username).
- Set permission level to **Viewer** or **Read-Only**.
- Confirm and apply the settings.

---

## Task 3: Verify Permissions

### Step 3.1: Test Dashboard Access
- Open a new browser or incognito window.
- Paste the shared link.
- Confirm access with read-only permissions.

### Step 3.2: Confirm Read-Only Access
- Attempt to edit a dashboard component to verify edit restrictions.
- Optionally, have the recipient confirm they can view but not edit.

---

## Conclusion
By completing this lab, you learned to:
- Export dashboards to PDF and CSV for offline use.
- Share dashboards with controlled read-only access.
- Safely distribute dashboards while protecting the original data and layout.
- Apply these skills to efficiently provide insights to stakeholders using open-source tools.
